### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sainip1atwit/Blankhouse2.git
   ```
2. Install NPM packages
   ```sh
   npm install multer pg bcrypt jsonwebtoken express
   ```
3. Start the server.js
   ```js
   cd src/services/
   node server.js
   ```
